## [0.0.5] - 2025-01-20

### 🎉 正式发布
这是 **Lyfe's Doc Editor** 的首个正式版本，专为中文开发者设计的智能 Markdown 编辑器。

### ✨ 核心功能

#### 📝 智能编辑体验
- **实时预览编辑器** - 基于 CodeMirror，支持 Markdown 语法高亮
- **智能格式切换** - 重复应用格式时自动切换或移除，避免格式嵌套问题
- **完整快捷键体系** - 支持标题、加粗、斜体、代码块等快捷操作
- **数学公式渲染** - 基于 KaTeX 的 LaTeX 公式支持
- **代码语法高亮** - 支持多种编程语言的语法高亮

#### 📱 多平台发布支持
- **微信公众号样式** - 一键适配微信公众号发布格式
- **知乎专栏样式** - 优化的知乎文章排版和样式
- **掘金技术社区** - 适配掘金平台的技术文章样式
- **GitHub Markdown** - 标准的 GitHub 风格样式
- **自定义主题** - 支持 CSS 自定义样式编辑

#### ☁️ 云端同步与存储
- **多设备同步** - 文档实时同步到服务器，支持多设备访问
- **智能自动保存** - 检测文档变更，每30秒自动保存
- **离线编辑支持** - 网络断开时可继续编辑，恢复连接后自动同步
- **数据安全保护** - 用户数据隔离，防止数据混淆

#### 📁 完整文档管理
- **文件夹组织** - 支持文件夹分层管理，拖拽排序
- **批量操作** - 批量保存、导出、删除等操作
- **回收站功能** - 误删文档可恢复，支持永久删除
- **全文搜索** - 快速检索文档标题和内容
- **文档版本管理** - 保留编辑历史，支持版本对比

#### 📲 文件导入导出
- **多格式导入** - 支持 Markdown(.md)、文本(.txt)、JSON(.json)、HTML(.html/.htm) 格式
- **批量导入** - 一次性导入多个文档文件
- **完整导出** - 导出包含元数据的完整文档数据
- **格式转换** - HTML 自动转换为 Markdown 格式

#### 🖼️ 图片管理系统
- **拖拽上传** - 支持图片拖拽上传，自动生成访问链接
- **多种图床支持** - 支持阿里云OSS、七牛云、GitHub等图床
- **图片格式支持** - 支持 JPG、PNG、GIF、WebP、SVG 格式
- **自动优化** - 图片自动压缩和格式优化

## 📝 版本说明

### 版本号规则
本项目采用 `主版本号.次版本号.修订版本号` 的版本控制方式：

- **主版本号**: 架构重大变更或不兼容的功能变更
- **次版本号**: 新增功能或重要功能改进
- **修订版本号**: 问题修复和小幅改进

### 发布周期
- **主版本**: 6-12个月发布一次
- **次版本**: 1-3个月发布一次  
- **修订版本**: 根据需要随时发布

### 支持策略
- **当前版本**: 长期维护和功能更新
- **前一版本**: 安全更新和重要修复
- **历史版本**: 安全修复（如有严重漏洞）
